# sorted

- sorted()를 할 때 key와 lambda를 사용하여 정렬할 수 있다.

```python
a = [[1, 1], [1, -1], [1, 13], [2, 2], [3, 13], [3, 4], [3, 3]]
print(sorted(a))
>>> [[1, -1], [1, 1], [1, 13], [2, 2], [3, 3], [3, 4], [3, 13]]

# x[0]을 기준으로만 정렬하고 있다. x[1]의 값은 정렬 x
print(sorted(a, key=lambda x: x[0]))
>>> [[1, 1], [1, -1], [1, 13], [2, 2], [3, 13], [3, 4], [3, 3]]

# x[1]을 기준으로만 정렬하고 있다.
print(sorted(a, key=lambda x: x[1]))
>>> [[1, -1], [1, 1], [2, 2], [3, 3], [3, 4], [1, 13], [3, 13]]

# sorted만 한 것과 값이 같다.
print(sorted(a, key=lambda x: (x[0], x[1])))
>>> [[1, -1], [1, 1], [1, 13], [2, 2], [3, 3], [3, 4], [3, 13]]

# x[1]의 값들로 먼저 정렬 된 후, x[0]의 값들로 정렬된다.
# 먼저 정렬된 기준이 더 큰 힘을 가진다.
print(sorted(a, key=lambda x: (x[1], x[0])))
>>> [[1, -1], [1, 1], [2, 2], [3, 3], [3, 4], [1, 13], [3, 13]]
```
